CSS


1. 태그를 추가하여 글자 색 변경 - < font color="색"> 내용 < /font >

2. 주석 처리 - < !-- 내용 -- >

3. style 태그 이용하여 글자 색 변경 - head 내에 style 태그 삽입 < style > 태그 { color : 색; } < /style >
< style > li { color: orange; } < /style > li 태그의 텍스트 컬러를 주황색으로 설정

4. html 태그 내에 CSS 스타일 삽입 (인라인 스타일) - 태그 내에 style = "속성:값"
여러 속성과 값을 넣고 싶을 경우 세미콜론으로 구분하여 이어적기

5. CSS 문법 구조 - 선택자(Selector) 효과(Declaration){ 속성(Property) : 값(Value); }

6. 선택자

- .클래스 이름 : class="클래스 이름"인 태그 선택
여러 번 사용하는 css가 있을 때 사용
태그 선택자 보다 우선 순위가 더 높음
우선 순위는 가장 최근에 선언된 것이 우선 순위가 높음

- #아이디 이름 : id="아이디 이름"인 태그 선택
한 번만 사용하는 css가 있을 때 사용
클래스보다 우선 순위가 더 높음

- 선택자를 두개 이어쓰면 첫번째가 부모고 뒤에가 자식으로 부모 태그 내에 있는 자식 태그를 선택한다는 의미

- 선택자의 우선 순위 : 태그 < 클래스 < 아이디
단, 같은 선택자끼리는 가장 최근에 선언 된 것이 우선 순위가 더 높음

7. 박스 모델

block level element : 화면 전체를 쓰는 태그
h, div, p, li, section

inline level element : 자신의 부피만큼만 쓰는 태그
a, span, img, em, i, strong

display : block / inline -> 블록 레벨을 인라인으로 바꾸거나 인라인을 블록으로 바꿀 시 사용

8. 그리드
- 페이지를 여러 주요 영역으로 나눌 때 사용
- display : grid / grid-template-columns / rows 등
- 1fr 1fr : 반반 / 2fr 1fr : 2 대 1

9. 반응형 웹 - 미디어 쿼리
- 화면의 크기에 따라 디자인 변경
- @media(min-width : 크기) { 태그 { css } }
