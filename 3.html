Javascript

1. 반복문
- while(true | false) { 실행할 코드; } 실행할 코드;
- while (조건) { 실행할 코드; } 실행할 코드;

- ex

            let i =0;
            while(i < 3){
              document.write('f');
              document.write('h');
              i += 1;
            }
            document.write('g');

2. 배열에 있는 값을 반복문을 활용해서 가져오기

            const arr = ['a', 'b', 'c', 'd'];
            let i = 0;
            while (i < 4) {
              // 조건문에 i < 4 대신 배열.length를 사용하면 배열의 길이가 바뀌어도 로직을 수정할 필요가 없다
              document.write('< li >'+arr[i]+'< /li >');
              i += 1;
            }


3. 함수

            function two() {
              document.write('f');
              document.write('g');
            }
          


            function sum(left, right) {
              document.write(left + '+' + right + '=' + left+right+'
');
            }
            sum(2,3);
            sum(9, 10);
- 해당 코드는 파라미터를 문자열로 처리함
- 왜냐하면 + 연산자가 문자열 연결에 사용되기 때문에 파라미터를 문자열로 연결하게 됨
- 따라서 결과가 2+3=23 / 9+10=910을 출력하게 됨 


            function sum(left, right) {
              document.write(left + '+' + right + '=' + (left+right)+'
');
            }
            sum(2,3);
            sum(9, 10);
- 해당 코드는 위 코드와 달리 파라미터를 숫자로 처리함
- 따라서 결과가 2+3=5 / 9+10=19를 출력하게 됨
- 출력 시 주의해야 함

4. 매개변수(parameter)와 인자(argument)
- sum(2,3)에서 2와 3은 인자
- sum(left, right)에서 left와 right은 매개변수라고 함
- 따라서 함수는 인자를 받아서 매개변수에 전달
- 이를 이용하여 함수를 실행한 후 결과를 출력하거나 반환

5. 객체
- 객체를 쉽게 설명하자면 이름이 있는 정리정돈 상자라고 할 수 있음
- 객체에는 배열, 문자열, 숫자, 불리언, 함수 등 다양한 데이터를 담을 수 있음 

            // 객체 생성
            const songs = {
              "lauv" : "I like me better",
              "lany" : "Anything 4 u"
            };
            // 값 가져오기
            document.write('lauv : ' + songs.lauv + "< br >");
            document.write('lany : ' + songs.lany + "< br >");
            // 추가 방법 1
            songs.coldplay = "Ink";
            document.write('coldplay : ' + songs.coldplay + "< br >");
            // 추가 방법 2
            songs["monsta x"] = "Middle of the night";
            document.write('monsta x : ' + songs["monsta x"] + " < br >");

6. 객체와 반복문
            // 속성 명 가져오기(key)
            for(var key in songs){
              document.write(key + '< br >');
            }
            // key는 lauv, lany 같은 값으로 속성 명에 해당하며 배열에서는 인덱스로 불림
          
            // 속성 값 가져오기(value)
            for(var key in songs){
              document.write(songs[key] + '< br >');
            }
          
            //속성 명 : 속성 값 형태로 가져오기(key : value)
            for(var key in songs){
              document.write(key + ' : ' + songs[key] + '< br >');
            }
